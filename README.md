# Getting started

You can install this theme by downloading the package on [Github](https://marketplace.visualstudio.com/items?itemName=Equinusocio.vsc-material-theme).

## Installation

To install the theme, download the theme & cd to location where you saved the extension

and run the following command

    code --install-extension /path_to_extension/g-dark-theme-0.0.1.vsix
